## Hoe te gebruiken

1. Open `index.html` in je webbrowser
2. Voeg een spel toe door de naam in te vullen en op "Spel Toevoegen" te klikken
3. Klik op "Speelsessie Toevoegen" om een nieuwe speelsessie te registreren
4. Klik op "Details Bekijken" om alle speelsessies voor een spel te zien
5. Je data wordt automatisch opgeslagen in localStorage
